시각장애인을 위한 VoiceOver(화면 읽기) 지원

복음서듣기는 iPhone의 VoiceOver를 지원합니다. 화면을 보기 어려우셔도 복음·장 선택, 재생·정지, 수면 타이머를 음성 안내와 터치만으로 이용하실 수 있습니다.

복음 이름·장 제목·재생 상태·경과 시간을 말로 읽어 주며, 재생 시작·정지·복음 변경·타이머 설정 시에도 음성으로 알려 드립니다. 말씀을 듣는 데 맞춘 앱이라 시각장애인 분께 특히 잘 맞도록 설계했습니다.

사용 방법: iPhone 설정 → 손쉬운 사용 → VoiceOver를 켠 뒤 앱을 실행하세요. 시리·단축어 · Voice Control · 햅틱 등 손쉬운 사용 기능 · 지원 페이지를 참고해 주세요.

손쉬운 사용 · 재생 편의

  • 햅틱 피드백: 재생·정지·장이 바뀔 때 짧은 진동으로 상태를 확인합니다.
  • Voice Control: 「재생 탭」「마태오 탭」처럼 음성만으로 버튼을 누를 수 있습니다.
  • VoiceOver 제목(Rotor): 「복음 선택」「장 목록」「재생 제어」로 구역을 나누어 탐색이 빠릅니다.
  • 이어 듣기 제안: 앱을 다시 열면 「이어서 마태오 5장 재생」을 한 번 탭해 이전 위치에서 재개합니다.
  • 잠금 화면·제어 센터: 재생 중 잠금 화면에서 이전 곡·다음 곡으로 장을 넘길 수 있습니다.

앱 소개

마태오·마르코·루카·요한 복음을 장별로 선택해 들을 수 있습니다. 이동 중이나 잠들기 전처럼 성경을 펼치기 어려울 때도 말씀에 귀 기울일 수 있도록 단순한 화면과 재생에 집중했습니다.

시리와 단축어로도 이용할 수 있습니다. 앱을 한 번 실행한 뒤 「마태오 3장 재생」, 「이어서 재생」, 「수면 타이머 30분」처럼 말로 재생을 시작하거나 멈출 수 있습니다.

시리 · 단축어로 말로 실행

App Intents를 지원합니다. 아래처럼 말하면 앱이 해당 동작을 수행합니다. 처음 사용 전에 앱을 한 번 실행해 주세요.

말하기 예시 동작
복음서듣기에서 마태오 3장 재생 해당 장 재생
이어서 재생 (앱 이름 포함) 정지 위치에서 이어 듣기
수면 타이머 30분 30분 후 자동 정지
복음서듣기 정지 재생 정지

60·90·120분 타이머는 단축어 앱에서 「수면 타이머 설정」을 추가해 선택할 수 있습니다. 자세한 안내는 지원 페이지를 참고해 주세요.

주요 기능

  • VoiceOver 지원: 복음·장·재생·수면 타이머를 음성으로 탐색·조작
  • 시리·단축어: 말로 장 재생, 이어 듣기, 수면 타이머, 정지
  • Voice Control: 「재생 탭」「마태오 탭」 등 음성 명령으로 조작
  • 햅틱·이어 듣기 제안·잠금 화면 이전·다음 장 (자세히)
  • 4복음서중 원하는 복음과 장 선택 후 재생
  • 선택한 장부터 복음 끝까지 연속 재생
  • 지금 재생 중인 장을 목록에서 쉽게 확인
  • 정지 후 재생 시, 가능한 경우 이전에 듣던 위치에서 이어 듣기
  • 수면 타이머: 잠자리에서 말씀 듣기에 맞춰 30·60·90·120분 뒤 자동으로 멈추거나, 밤새 들으려면 ‘계속’을 선택하세요. 남은 시간이 화면·음성 안내로 표시됩니다.
