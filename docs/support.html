시각장애인 · VoiceOver(화면 읽기) 사용 안내

복음서듣기는 Apple VoiceOver를 지원합니다. 화면을 보지 않고도 아래 기능을 음성 안내로 이용할 수 있습니다.

  • 복음 선택: 마태오·마르코·루카·요한 버튼 이름과 선택 상태 읽기
  • 장 목록: 각 장 제목, 재생 중·선택됨 상태, 경과·전체 재생 시간(말로 안내)
  • 재생·정지: 재생·정지 버튼, 정지 후 이어 듣기 안내
  • 수면 타이머: 타이머 설정·남은 시간 음성 안내
  • 상태 알림: 재생 시작·정지·복음 변경·타이머 변경 시 음성 알림

켜는 방법

iPhone에서 설정 → 손쉬운 사용 → VoiceOver를 켠 뒤 앱을 실행하세요. 한 손가락으로 화면을 오른쪽으로 밀면 다음 항목을, 두 번 탭하면 선택합니다.

VoiceOver 사용에 익숙하지 않으시면 Apple의 iPhone VoiceOver 사용 설명서도 참고해 주세요.

자주 묻는 질문

소리가 나오지 않아요.

기기 음량·무음 스위치·다른 앱 재생 상태를 확인해 주세요. 설정에서 마이크가 아닌 재생 볼륨을 조정해 보세요.

VoiceOver로 어떻게 쓰나요?

iPhone 설정 → 손쉬운 사용 → VoiceOver를 켠 뒤 앱을 실행하세요. 복음·장·재생·수면 타이머가 모두 음성으로 읽히며, 재생·정지 시에도 알림이 들립니다. 위 VoiceOver 안내를 참고해 주세요.

정지 후 다시 재생하면 처음부터 나와요.

정지 시점의 장과 위치를 기억합니다. 다른 복음을 둘러본 뒤 재생을 누르면 정지했던 장에서 이어 들을 수 있습니다. 목록에서 다른 장을 탭하면 그 장부터 새로 재생됩니다.

수면 타이머는 어떻게 동작하나요?

화면 상단의 수면 타이머 버튼에서 원하는 시간(30·60·90·120분)이나 ‘계속’을 고릅니다. 타이머는 재생 중일 때만 만료 시 자동으로 멈추며, ‘계속’이면 밤새 재생됩니다. 남은 시간은 제목 옆에 표시됩니다.

iPad나 Mac에서도 되나요?

프로젝트 설정에 따라 지원 대상 기기가 달라질 수 있습니다. App Store의 ‘정보’에서 호환 기기를 확인해 주세요.
